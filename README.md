### Como Rodar

com o pip instale o pipenv

depois na pasta raiz do projeto sete suas variaveis de ambiente no .env
para criar o .env basta copiar o .env.example

```bash
cp .env.example .env
```

Depois instale os requisitos

```bash
pipenv install && pipenv install -d
```
